# Round 1 — Vision

Date: 2026-05-15
Status: LOCKED (user-directed reference port)

## 1. Pillar fantasy
Player feels like a lucky scavenger pushing coins off a chaotic ledge — every run a fresh tumble of physics, jackpots, and roguelike upgrades that compound across sessions.

## 2. Reference games
- **RACCOIN: Coin Pusher Roguelike** (primary donor)
  - ✅ Copy: coin-pusher physics core, roguelike upgrade picks between runs, escalating jackpot/bonus chains, "one more push" tactility, modifier-stacking depth
  - ❌ Drop: PC-only UI density, premium-priced one-shot loop, lack of meta retention, deck-builder hard ceiling
- **Coin Master** (F2P comparable)
  - ✅ Copy: daily-energy gating, social viral hooks, sticker/collection meta, push notification re-engagement
  - ❌ Drop: PvP raid griefing, slot-machine reskin, weak skill expression
- **Balatro** (run-economy comparable)
  - ✅ Copy: synergistic modifier picks, ante/run structure, "just one more run" pacing, joker-style modifier UX
  - ❌ Drop: card-game theme, premium pricing, hard difficulty wall

## 3. Audience + platform
- Mobile F2P (iOS + Android)
- Primary: casual-to-mid-core players who like coin pushers, idle/casino-adjacent games, and roguelite progression
- Secondary: Coin Master / Monopoly Go / Balatro crossover audience

## 4. Hard constraints
- ✅ Must-have: mobile portrait, F2P monetization (energy/lives + IAP shop + ads), short sessions (90-180s per push run), roguelike upgrade meta, persistent collection/progression
- 🚫 Must-not-have: PvP raids that punish offline players, paywalled core verb, complex tutorial chains
