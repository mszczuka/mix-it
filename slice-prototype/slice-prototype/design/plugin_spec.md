# Plugin spec — Slice Tycoon

Date: 2026-06-26
Archetype: ActiveTapperIdle (NEW)

## Name
Slice Tycoon (chosen by user; original, not derived from project dir).

## Core verb
Tap-to-slice: single tap on the served ingredient slices it; tap timing in a
"perfect window" feeds a combo multiplier. One-thumb, portrait.

## Win condition
Progression milestones — each ingredient/blade upgrade is a local win; the first
prestige ("Master Knives" reset → permanent global multiplier) is the headline win.

## Lose condition
Soft-fail only: knife-wear throttles income (worn blade slices slower) until the
player sharpens/upgrades. Never a game-over.

## Prototype slice (MVP — accepted scope)
tap-to-slice + combo + Coins + ONE blade upgrade + knife-wear + offline earnings +
first prestige. Cut for phase-2: decorations, full ingredient tree, automation
managers, daily/weekly/seasonal, real ads/IAP, season pass. These appear as
visible-but-stubbed UI to show direction.
