# Round 3 — Meta (apprentice proposal)

| Field | Proposed | Alternative | Rationale |
|---|---|---|---|
| Currencies (soft) | Coins — per slice + offline; spend on ingredients, blades, decorations, global upgrades | Two soft currencies (coins + prep tickets) | One soft currency keeps it one-thumb-simple; dual adds tutorial load (MUST-NOT). |
| Currencies (hard) | Gems — IAP-bought, trickle via milestones; permanent 2x, time warps, auto-unlocks, ad-removal | No hard currency, pure ad-gated | Dual currency = F2P backbone; gems enable IAP catalog without forcing ads (avoids Idle Slice & Dice anti-pattern). |
| Currency (prestige) | Master Knives (angel-equiv) — earned on prestige; permanent global profit multiplier | Golden Chefs passive producers | Maps to AdCap angels + slicing fantasy. Naming TBD. |
| Progression tracks | (1) Ingredient tiers veg→fruit→meat→exotic (2) Blade tree unlock+upgrade+sharpen (3) Kitchen decorations w/ bonuses (4) Automation managers (auto-slicers) (5) Prestige/ascension layers | Fewer tracks (blades + prestige only) | Mirrors StMY features + AdCap automation; parallel tracks create upgrade choices. Staggered unlock. |
| Automation | Managers/auto-slicers as milestone rewards; auto-slice reduced vs active (active 2-5x better) | Auto-slice from day 1 | Active-tapper-idle spine; day-1 automation makes MUST active verb pointless. |
| Offline earnings | On at install; ~50-100% active rate, capped (8h = 1-2h active eq.); generous welcome-back | Uncapped offline | Safety net + return trigger; cap drives daily check-ins. |
| Retention — daily | Daily login reward, offline-cap "your board is full!" notif, daily-deal blade/decor, 1-2 slice quests | Heavy daily-quest checklist | Light touch fits cozy + 2-5 min session; chore-heavy contradicts anti-stress. |
| Retention — weekly | Weekly ingredient/blade challenge w/ milestone track; gentle streak bonus (no harsh reset) | Weekly competitive leaderboard | Reward consistency without punishing absence; leaderboard deferred (single-player). |
| Retention — seasonal | Themed live-ops events (seasonal ingredients/decor, limited blades), optional Season/Event Pass ($4.99-9.99) | Hard FOMO limited banners | Idle needs events for long lifecycle; soft scarcity keeps urgency mild vs cozy tone. |
| Monetisation stance | Hybrid, ad-light & opt-in — rewarded video as bonus ("watch to 2x offline / +profit 30min"); IAP = permanent 2x, time warps, auto-unlocks, ad-removal, starter bundle; NO forced interstitials mid-slice, NO hard paywalls | Ad-heavy IAA (Idle Slice & Dice style) | Resolves cozy-vs-F2P; opt-in rewarded = highest-completion, natural in waiting mindset. Completable without ads. |
| Monetisation hooks | First-purchase starter bundle; permanent 2x; time-warp (1/4/8/24h, small via ad); prestige booster; ad-removal $2.99-4.99 | Subscription-first | One-time IAP + opt-in ads suit broad casual + cozy; subscription-only ad-removal = the anti-pattern. |
