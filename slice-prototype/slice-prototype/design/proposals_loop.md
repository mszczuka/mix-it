# Round 2 — Loop shape (apprentice proposal)

| Field | Proposed | Alternative | Rationale |
|---|---|---|---|
| session_length | 2-5 min active check-ins, 5-8 sessions/day | Single 10-15 min "lean-back" session | Matches idle.md (2-5 min, 3-6/day); short habitual check-ins are by design. |
| core_verb | Tap-to-slice (tap/swipe on served ingredient; timing feeds a combo multiplier) | Hold-to-auto-slice from start | MUST = active tap, one-thumb. Combo keeps verb meaningful after automation. Alt collapses to pure-AFK. |
| win_cond | Progression milestones: first prestige, then ascension layers; each ingredient tier/blade/kitchen = local win | Single fixed "beat the game" endpoint | Idle has no PvP win; layered milestones + prestige sustain 30-90 day lifecycle. |
| lose_cond | Soft-fail via knife-wear: worn blade slices slower/chokes on hard produce, throttling income until sharpen/upgrade — never game-over | Hard fail: blade breaks, run resets | Knife-wear is a MUST + keeps tap strategic. Soft-fail preserves cozy/no-pressure pillar. |
| mode | Single-player active-tapper idle (offline accumulation between sessions) | Add async social (ghost leaderboards) | Constraints forbid realtime PvP; keep core pure. Async leaderboards = later live-ops (parked). |
